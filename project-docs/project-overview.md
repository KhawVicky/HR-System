# Project Overview

## System Name

AI-Based Human Resource Decision Support System for UWC Berhad.

## Purpose

The system helps UWC Berhad HR staff and hiring managers process recruitment faster, more consistently, and more transparently.

It supports:

- Uploading job descriptions.
- Reviewing auto-filled job details.
- Generating unique application links.
- Receiving candidate applications.
- Evaluating resumes.
- Ranking candidates.
- Showing score breakdowns.
- Supporting shortlist, interview, and reject decisions.

The system does not automatically decide who should be hired. Final decisions remain with UWC Berhad staff.

## Core Users

| User | Access |
| --- | --- |
| HR Staff | Create jobs, upload JD, review candidates, shortlist, send interview or reject email |
| Hiring Manager | Review shortlisted candidates, view score breakdown, manage users |
| Candidate | Access only the application page through a shared job link |

## Core Problem

Current resume screening can be slow, inconsistent, hard to explain, and scattered across email. HR needs a centralised system to manage jobs, applications, candidate ranking, and score explanations.

## Core Scope

- Job Description Management
- Application Link Management
- Candidate Application Management
- Resume Evaluation and Ranking
- Score Breakdown
- Candidate Review and Shortlisting
- Interview and Reject Actions
- User Profile, Security, and Notification
- User Management for Hiring Manager
- Attendance Analytics as optional feature

## System Flow

1. Internal user logs in.
2. HR views dashboard.
3. HR creates a job and uploads JD.
4. System auto-fills job details.
5. HR edits job details.
6. HR sets criteria, weights, and eligibility filters.
7. System generates unique application link.
8. Candidate applies through application link.
9. Candidate uploads resume.
10. System parses resume and evaluates candidate.
11. System calculates score and eligibility status.
12. HR views candidate ranking.
13. HR opens candidate details and score breakdown.
14. HR shortlists, interviews, or rejects candidate.
15. Hiring manager reviews shortlisted candidates.

## Key Differentiator

Normal ATS tools mainly track applications. This system focuses on decision support through custom criteria scoring, weighted ranking, eligibility filtering, and transparent score breakdown.
